Portfolio

Released Projects

Realm of Rulers

Realm of Rulers is an early access Multiplayer Medieval Fantasy City Building game with over 82,700 downloads, 234,000 page views and over 1.7 million Youtube views.

You are the baron of a new town and your job is to manage the town's resources and population to try and create a world that will control itself through NPC interactions, AI gathering, refining and construction. The game is in development by Mathew Hill and Lewis Taylor.

Catch

Catch is a free mobile android game made using the Unity engine. You have to catch orbs that fly towards your collector. Every orb you collect gives you points which go into your online global leaderboard score using the Google Play games services.

There are also unlockable achievements through Google Play which adds incentive to get the best score possible.

University Projects

Mathew's Mod

Mathew's Mod is an Unreal Engine 4 recreation of the game Garry's Mod. This was done for the first semester of my second year at university to learn and make a simple game using Unreal Engine C++. The assignment brief only required a small prototype but as I strive to push myself and my abilities I recreated a fully working demo which contains the tools and mechanics to make simple vehicles and structures.

College Projects (2015/2018)

Hide and Seek

Hide and Seek was a two month project that I created during the summer holidays between the first and second year of college. The core gameplay is a first person multiplayer horror game where each player is put into one of two teams, hiders and seekers.

The main features of the game are a fully working multiplayer game with lobby interface with map select, chat system, admin capabilites (Kick and Ban players from the game.), a server browser where playable matches show up with a simple search and a class based multiplayer game mode with map selection

Retribution

Retribution is a top down shooter created created in 2 months for an assignment at my first year of college. I was required to make a 2D game demo that was interactive and had at least 1 level.

I ended up having enough time to create 5 levels with unique music and sound effects created by myself. This was the first project I had created using the Unreal Engine.

The Arena

Our first second year assignment was to create a 3D level for a first person shooter game that was based somewhere in our college. We researched maps in existing games such as Counter Strike and Call of Duty to understand what makes a level fun to play.

The assignment brief did not require multiplayer functionality but testing with others was hard unless we played together so I decided to learn the multiplayer functionality that Unreal Engine 4 has and implement this into the level.

From there we tested and refined the level by playing and finding areas to improve on and through the course of 3 months the shooting, score, design and functionality was fully complete.

Unbound

For our final second year college project we were tasked to create a 3D game with narrative elements using Unreal Engine 4. We were put into teams with each member having their own role such as level design, 3D art and 2D art.

Together we had a meeting to decide the genre, setting, gameplay mechanics and a basic version of the storyline. The assignment brief required us to have at least one cutscene which advances the storyline. The group and I decided that we wanted to create the best most complete game we could while still remaining on target for our deadline.

Castle Horror College Game Jam 2016

This game was for the 2016 college game jam in which Archie (2D and 3D Artist), Josh (Level Designer) and I (Programmer) had 48 hours to create something fun using Unreal Engine 4.

The objective is to collect three keys to escape the castle while dodging alien creatures using your hearing and sight. We designed ideas and mechanics as fast as possible and after only 8 hours we had finished the game.
